# hokkaido-place-quiz
# **北海道地名読みクイズ**

北海道の地名を題材にした、**読み方を入力して答えるクイズアプリ**です。

Vue 3 + TypeScript と Rails API を用いて、フロントエンドとバックエンドを分離した構成で開発しています。

---

## **🎯 アプリ概要**

- 北海道の地名（漢字）が出題される
- ユーザーは読みを **テキスト入力** して回答
- 正誤判定を行い、10問終了後に結果を表示
- 結果は **X（旧Twitter）でシェア可能**

---

## **✅ MVP機能**

### **① クイズ出題（入力式）**

- 北海道の地名（漢字）を1つ表示
- ユーザーが **読みをテキスト入力**
- ひらがな入力前提
- クイズの流れ
    
    **地名表示 → 回答入力 → 判定表示 → 次の問題へ**
    

---

### **② 正誤判定**

- 入力値と正解の読みを比較
- 正解 / 不正解を判定
- 不正解時は **正しい読みを表示**

※ 判定は「完全一致」のみ

（表記ゆれ対応は行わない）

---

### **③ 問題数：10問**

- 1プレイ10問
- ランダム出題
- 同じ問題は出題されない

---

### **④ 進行表示**

- 現在の問題数 / 全体数を表示
    
    例：3 / 10
    
- 「次の問題へ」ボタンで進行

---

### **⑤ 結果画面**

- 正解数を表示
    
    例：7 / 10
    
- 簡単な結果メッセージ
    
    例：「なかなか道民レベルです！」
    

---

### **⑥ X（旧Twitter）シェア**

- クイズ結果をテキストでシェア可能

---

## **🗄 ER図（DB設計）**

### **places テーブル**

| **カラム名** | **型** | **必須** | **説明** |
| --- | --- | --- | --- |
| id | bigint | ○ | 主キー |

## **🛠 使用技術**

### **フロントエンド**

- **Vue 3**
    
    JavaScriptフレームワーク。UI（画面）構築を担当。
    
    状態管理が多いクイズアプリと相性が良い。
    
- **TypeScript**
    
    JavaScriptに型を追加した言語。
    
    型安全性と可読性を向上。
    
- **Vite**
    
    フロントエンドの開発・ビルドを行うツール。
    
    - .vue / .ts を JavaScript に変換
    - ホットリロード対応
    - 本番ビルドの最適化

---

### **バックエンド**

- **Ruby 3.4.5**
- **Ruby on Rails 8.0.2.1（APIモード）**
    
    JSONを返却するAPI専用のバックエンドとして使用。
    

---

### **データベース**

- **PostgreSQL**

---

### **インフラ**
