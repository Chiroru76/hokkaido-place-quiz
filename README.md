## 📝 概要
北海道にある 179市町村 の地名の読み方を答える、クイズ形式のWebアプリです。

北海道は、日本で 最も市町村数が多い都道府県（2位：長野県 77市町村）であり、
アイヌ語由来の 難読地名 も多く、北海道民でも読み方に迷う地名が数多く存在します。

本アプリでは、地名クイズを通して
「北海道の地理」を学ぶことができるだけではなく「各市町村の特徴」や「観光スポット」
についても楽しみながら知ることができます。

**🎯 目指せ、179市町村完全制覇！ 🏆**

## 🎮 機能一覧

### 🗺 市町村の詳細情報を確認
[![Image from Gyazo](https://i.gyazo.com/1a7a14a92c8588cee705e6824e3dfd10.gif)](https://gyazo.com/1a7a14a92c8588cee705e6824e3dfd10)
クイズの正誤判定後に、出題された市町村について詳しく知ることができます。
	•	地域に関する基本情報・豆知識を表示
	•	Google Map 上で市町村の位置を確認
	•	東京駅からの 所要時間・距離 を表示（Routes API）
	•	周辺の 観光スポット を一覧表示（Places API）
👉 クイズをきっかけに、その地域への理解を深められます。

## 💡 工夫した点

### 🗾 達成状況を可視化するマップ表現

正解した市町村を地図上で 色分けして表示 することで、
どこまでクリアできたのかを一目で確認できるようにしました。
	•	正解済みの市町村が視覚的に分かる
	•	「すべて色を塗りたい」という 達成欲求 を自然に喚起
👉 継続的に遊んでもらえるような設計を意識

### 📍 地名クイズ＋αの情報による楽しみ
正誤判定後に、単に「正解／不正解」を表示するだけでなく、
	•	地域の豆知識
	•	地図上での位置確認
	•	アクセス情報や周辺の観光スポット

などを表示することで、一問一答形式のクイズにとどまらない楽しみ方 ができるようにしました。
	•	「この地域には何があるのか？」という興味を引き出す
	•	他の市町村や観光地にも自然と関心が広がる
👉 クイズをきっかけに、地域への理解を深めてもらう
